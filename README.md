# Show Not Tell — Landing Page

Client project built for [Show Not Tell](https://shownottell.studio), a visual storytelling agency based in Bremen, Germany.

## What it is
A marketing landing page featuring a personalized video message, client testimonials, a 3-step process section, FAQ, and a booking page with Calendly integration.

## Pages
- `/` — Landing page with embedded Wistia video, testimonials, process, and FAQ
- `/booking` — Calendly embed for scheduling a free 30-min consultation

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS (via CDN)
- React Router (client-side navigation)
- Wistia (video embed)
- Calendly (booking widget)
- Google Analytics (GA4) + Meta Pixel (conversion tracking)

## Run Locally

```bash
cd my-app
npm install
npm run dev
```

App runs at `http://localhost:5173`

## Build for Production

```bash
npm run build
```
